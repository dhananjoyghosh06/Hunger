Intro: 
It is a Fullstack M.E.R.N food ordering delivery system web application 

TECH STACK : 

Frontend : HTML5, CSS (TAILWIND), JAVASCRIPT, REACT.JS, Bootsrap

Backend : JAVASCRIPT, NODE.JS, EXPRESS.JS

Database : MongoDB

Others : Cloudinary, Multer, React-icons, Redux-toolkit, React-router-dom, Twillo 

Description :
1. Frontend development using React.js for creative , responsive interface, Backend development using Node.js and Express.js and to store and manipulate data used MongoDB
2. User Auth : this is a mobile  OTP based authentication user application and used JWT token and sessions 
3. Used and handled  REST APIs  using Node.js to get like Restaurant details, Cities details etc 
4. To fetch,  insert,  update , delete data I used Axios from frontend and backend too
5. Upload and update user profile image using muter and stored dynamically in Cloudinary
6. Used React-Redux for add-to-cart option for user and for payment used
